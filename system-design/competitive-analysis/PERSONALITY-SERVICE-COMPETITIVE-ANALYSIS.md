### 3.2 Profile Stability Scoring **UNIQUE**

**Implementation**: `services/personality_service/src/domain/service.py`

**Capabilities**:
- **Variance-Based Stability**: Measures consistency across assessments
- **Rolling Window**: Uses last 5 assessments for stability calculation
- **Confidence Adjustment**: Higher stability = higher confidence in profile

**Algorithm**:
```python
def _compute_stability(self, profile: PersonalityProfile) -> float:
    """Compute profile stability based on assessment variance."""
    if profile.assessment_count < 3:
        return 0.3  # Low stability until sufficient data

    variance_sum = 0.0
    for trait in PersonalityTrait:  # O, C, E, A, N
        values = [s.get_trait(trait) for s in profile.assessment_history[-5:]]
        mean = sum(values) / len(values)
        variance = sum((v - mean) ** 2 for v in values) / len(values)
        variance_sum += variance

    avg_variance = variance_sum / 5
    return max(0.0, min(1.0, 1.0 - avg_variance * 10))  # Lower variance = higher stability
```

**Competitive Assessment**: **UNIQUE** - No competitor publicly discloses stability scoring

**Clinical Relevance**:
- Personality is relatively stable but shows measurement variance
- Stability scoring identifies reliable vs. noisy profiles
- Enables confidence-weighted adaptation decisions

## Critical Competitive Learnings

### Features to Adopt (Cross-Service)

| Competitor | Feature | Priority | Applicable Services |
|------------|---------|----------|---------------------|
| **Spring Health** | VERA-MH ethics framework | HIGH | All services |
| **Headspace** | Ebb conversational AI with motivational interviewing | HIGH | Therapy, Orchestrator |
| **Limbic** | Class IIa CE certification pathway | HIGH | All clinical services |
| **Mem0** | Graph-enhanced memory (Mem0ᵍ) | MEDIUM | Memory |
| **Crisis Text Line** | Real-time supervisor dashboard | MEDIUM | Safety |
| **Microsoft Agent Framework** | MCP/A2A open standards | MEDIUM | Orchestrator |

### Differentiation Opportunities

1. **Multi-modality therapy** (CBT/DBT/ACT/MI/Mindfulness) - few competitors offer all
2. **4-layer crisis detection** - exceeds industry standard of single-method approaches
3. **5-tier cognitive memory** - unique Ebbinghaus decay model
4. **Personality-adaptive responses** - unique in mental health AI space
5. **Integrated clinical agents** - unique combination in LangGraph orchestration
